# Image-Classification-Using-Haar-Cascade
Wrote a Python program to detect faces in an image using Haar Cascade.

Used "haarcascade_frontalface_default.xml" provided by OpenCV to detect faces in an image.

## Libraries Used
1. cv2
2. numpy
3. opencv-utils
4. opencv-python
